### 개발 목표

동적으로 ToDo를 생성, 삭제한다.  
생성한 투두 리스트를 새로고침 이후에도 유지되도록 한다.

### 사용한 기술

HTML, CSS, JavaScript



### 성과

DOM을 조작하여 원하는 요소를 동적으로 추가, 제거할 수 있다.  
원하는 데이터를 유저의 LocalStorage에 저장하고 불러와서 사용할 수 있다.
